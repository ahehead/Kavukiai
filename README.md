## Kavukiai

Kavukiai(仮)は、主にローカルでAIを繋げて楽しむことができる、ノードUIのデスクトップツールです。

ノードをつなげて、AIワークフローを作ることができます。

## 特徴

- このツールことKavukiaiは、ComfyUI+プラグイン、Dify、n8n、(ChatGPTなどを使った)バイブコーディングなどの完全下位互換です。なので真面目な用途ではまずそちらを使ってみてください。説明をみてこちらのツールだと何か試せそうだと閃いたらKavukiaiを使ってみてください。
- Kavukiaiはノードエディタであり、LM StuidoやComfyUIを呼び出すノードがあり、それらを組み合わせてワークフローを作ることができます。
- 特徴としてノードに実行ボタンが付属していて、処理の流れも変えることができます。
- 実験的に作ったα版です。基本的なtext to imageのみ動作確認しています。
- ワークフローはJSONまたはPNGで保存できます。またそれらをドラッグアンドドロップで読み込むことができます

[LM Studio](https://lmstudio.ai/)と[Comfy UI Desktop](https://www.comfy.org/)をインストールして適当なモデルをダウンロードしてください。

例えば…

## インストール
zipをダウンロードして解凍します。そのあとexe等のファイルをクリックで起動します。
(現在証明書などがないので、警告がでます)

## 操作
- ノードは移動できます。
- コネクションは、処理の流れ(push)のオレンジとデータの流れ(pull)の青色があります。
- 右クリックメニューからノードの作成
- マウス中ボタンで移動
- ctrl + c , ctrl + v でノードのコピーとペースト。
