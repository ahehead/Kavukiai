## Kavukiai

Kavukiai(仮)は、主にローカルでAIを楽しむためのノードUIのデスクトップツールです。

ノードをつなげて、ワークフローを作り、実行することで、ローカルAIツール同士を接続して実行します。

## 特徴

- まず、このツールは、ComfyUI+プラグイン、Dify、n8n、ChatGPTで直接作るプログラム等の下位互換なので真面目な用途ではそちらを使ってください。説明をみてこちらのツールだとあれが試せそうだと何か閃いたら使ってみてください。
- Kavukiaiはノードエディタであり、LM StuidoやComfyUIを呼び出すノードがあり、それらを組み合わせてワークフローを作ることができます。
- 特徴としてノードに実行ボタンが付属していて、処理の流れも変えることができます。
- 開発版で今は基本的なtext to imageのみ動作確認しています。
- ワークフローはJSONまたはPNGで保存できます。またそれらをドラッグアンドドロップで読み込むことができます
